       功能介绍：一款智能问句识别与同义生成系统，主要用于文本信息处理、格式调整与内容筛选。程序首先利用句向量模型（SentenceTransformer）计算问句之间的语义相似度，对原始Excel中的问题进行**分类与标注**（standard/same/different）；随后调用Moonshot接口自动生成与锚点问题语义相近的新问句，并实时写入新的Excel文件中，可实现问句数据的批量扩充与结构化整理，适用于智能问答系统训练集构建、语料库优化及教育类知识问答整理等场景。

      主要场景示例：在传统自动客服场景中，本系统可以将历史用户问答导入Excel，自动识别并分类问题为标准化、同义或不同语义的问题，并通过Moonshot接口生成新的近义问句，实时写入Excel，从而快速扩充问句库，提高自动客服或智能问答系统对多样化用户提问的识别和响应能力。
      
     提示：推荐使用python.3.10,新版本可能存在句向量模型不兼容问题。
